Bootstrap

Bootstrap är ett av de mest populära CSS-ramverken att och går att inkludera antingen via nedladdning (t ex genom npm) eller via CDN.

Det baseras på både Sass som CSS och på JavaScript för att kunna dölja/visa element, till exempel för mobilnavigering.

Det består dels av klasser som kan sättas på vilka element som helst, men har också många färdiga komponenter såsom navbars, footers, accordions och så vidare.

Bootstrap har utförlig dokumentation på Bootstraps hemsida.

Fördelar

  • Man får responsiviteten på köpet när man använder Bootstraps container-klass, och likaså kan man få en responsiv mobilmeny som är redo att använda. All JavaScript finns redan på plats om man inkluderar en CDN-länk i en script-tagg.
  • Tabeller och formulär är lättare att lägga till eftersom även de är enkla att få responsiva och lite lättare att styla.
  • Det sägs att det sparar tid och det är ju smidigt i teorin att kunna smälla på lite klasser på taggarna bara.
  • Bra och utförlig dokumentation med flera exempel på anpassningar och olika användningar. Finns också ett cheatsheet som kommer väl till pass när man ska lära sig lite snabbt.
  • Vissa komponenter är faktiskt skönt att slippa skapa själv, till exempel accordions och progress bars.

Nackdelar

  • Man kan säkert anpassa Bootstrap mycket mer än vad man tror, men det känns inte enkelt. Jag tycker inte om begränsningen i valfrihet.
  • Jag har lagt långt mycket mer tid på att försöka läsa på hur man kan positionera saker än det hade tagit mig att bara styla sakerna helt själv med en egen css-fil.
  • Jag har aldrig upplevt formulär och knappar som särskilt jobbiga att ha att göra med, så det tilltalar inte mig särskilt mycket som säljpunkt att det skulle bli "jättemycket smidigare". Tabellen var dock skön att slippa.
  • Färgerna är svåra att påverka med CDN-inkluderad Bootstrap och man har otroligt begränsade val.
  • Det känns som att om man vill få det faktiskt snyggt och anpassat så kommer man behöva lägga en del tid och energi på det. Då känns det kanske nästan smidigare att bara skriva sin egen CSS på en gång?

Jag är kanske lite orättvis mot Bootstrap här. Jag är säker på att det finns många fördelar fler än jag har listat, och nackdelarna stämmer säkert in på många fler ramverk också.

Jag misstänker också att man kan påverka mer om man inte kör via CDN.
